Predicting Natural Gas Prices

Inspiration

With all the uncertainty lately in prices at the pump, this is also being mirrored in the natural gas market. We wanted to see if we could develop a machine learning model to predict price changes in natural gas based on a few variables.



These Variables being:

  • Node or Location
  • Volatily
  • Demand/Consumption
  • and Inventory

Objective

Create a machine learning model to predict natural gas prices in the future.
